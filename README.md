# Number Base Converter

A Python GUI application built using Tkinter that converts between:

- Binary ↔ Decimal
- Binary ↔ Hexadecimal
- Decimal ↔ Hexadecimal

## Features

- Simple graphical interface
- Error handling for invalid inputs
- Supports multiple base conversions

## Technologies Used

- Python
- Tkinter

## How to Run

1. Make sure Python is installed.
2. Download the file.
3. Run:

   python number_base_converter.py
